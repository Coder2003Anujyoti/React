# Kabaddi Game Repository 🏃‍♂️🤾‍♀️

Welcome to the Kabaddi Game Repository! This project is a digital version of the classic and thrilling game of Kabaddi, where two teams compete to score points by tagging opponents and avoiding being tagged. 🏅

## Features
- Realistic game mechanics 🏃‍♀️
- Scoring system ⏱️
- Player stats tracking 📊
- Interactive UI/UX 🎮


## How to Play
- **Raider:** Tag opponents while holding your breath 💨
- **Defender:** Avoid the raider and tag them before they escape 🚶‍♂️
- Each successful raid scores points for your team 🏆

## Technologies Used
- **Frontend:** React,CSS
